# Java-Advanced-Payment-Model
This project demonstrates a simple payment processing system implemented in Java using Object-Oriented Programming principles.

The system simulates a real-world payment gateway that supports multiple payment methods such as:

Credit Card

UPI

PayPal

The design uses interfaces and polymorphism, allowing easy extension for additional payment methods in the future.

Key concepts demonstrated:

Interface-based design

Polymorphism

Encapsulation

Clean payment workflow simulation

This code can serve as a basic foundation for building larger payment gateway integrations in enterprise Java applications.
